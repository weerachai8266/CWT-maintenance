# คู่มือการสร้าง QR Code สำหรับแจ้งซ่อมเครื่องจักร

---

## ภาพรวม

QR Code ที่สร้างจากระบบนี้ใช้สำหรับ **ติดที่ตัวเครื่องจักร** เมื่อพนักงานสแกน QR จะเปิดฟอร์มแจ้งซ่อมโดยอัตโนมัติ พร้อมกรอกข้อมูลเครื่องจักร (ฝ่าย, หน่วยงาน, สาขา, รหัสเครื่อง) ล่วงหน้าให้แล้ว

---

## ขั้นตอนที่ 1 — เข้าสู่ระบบ

1. เปิดเบราว์เซอร์แล้วไปที่ URL ของระบบแจ้งซ่อม
2. หน้าเข้าสู่ระบบจะปรากฏขึ้น
3. กรอก **ชื่อผู้ใช้** และ **รหัสผ่าน** ของ MT staff
4. กดปุ่ม **"เข้าสู่ระบบ"**

> **หมายเหตุ:** ทุก Role ที่มีบัญชีในระบบสามารถเข้าใช้งานหน้า QR Code ได้

---

## ขั้นตอนที่ 2 — ไปยังหน้า QR Code

หลัง login สำเร็จ มี 2 วิธีเข้าหน้า QR Code:

**วิธีที่ 1:** คลิกเมนู **"QR Code"** ที่ Navbar ด้านบน (มีไอคอน 🔲)

**วิธีที่ 2:** ไปที่หน้า **"เจ้าหน้าที่ MT"** แล้วคลิกเมนู **"QR Code"**

---

## ขั้นตอนที่ 3 — เลือกเครื่องจักร

หน้า QR Code แบ่งเป็น 2 ส่วน:
- **ซ้าย:** รายการเครื่องจักรทั้งหมด
- **ขวา:** แผง QR Code

### การค้นหาเครื่องจักร

| วิธี | รายละเอียด |
|------|-------------|
| พิมพ์ในช่อง 🔍 | ค้นหาด้วยรหัส, ชื่อ, หรือหมายเลขเครื่องจักร |
| Dropdown "สาขา" | กรองเฉพาะสาขาที่ต้องการ |

### การเลือกเครื่องจักร

- **คลิกที่การ์ด** เครื่องจักรที่ต้องการ  
  หรือ  
- กดปุ่ม **"📷 เลือก"** ที่ขวาของการ์ด

การ์ดที่เลือกจะมีกรอบสีน้ำเงิน และแผงด้านขวาจะแสดงข้อมูลเครื่องจักรนั้น

---

## ขั้นตอนที่ 4 — เลือกฝ่ายและหน่วยงาน

ในแผงด้านขวา:

1. เลือก **"ฝ่าย"** จาก dropdown
2. เลือก **"หน่วยงาน"** จาก dropdown

> **เคล็ดลับ:** ถ้าต้องการสร้าง QR สำหรับเครื่องจักรหลายเครื่องในฝ่าย/หน่วยงานเดียวกัน ระบบจะ **จำค่าที่เลือกไว้** และ auto-เลือกให้เมื่อเปลี่ยนเครื่อง ไม่ต้องเลือกซ้ำ

---

## ขั้นตอนที่ 5 — ดูตัวอย่าง QR Code

เมื่อเลือกฝ่ายและหน่วยงานครบแล้ว ระบบจะแสดง **ตัวอย่าง QR Code** โดยอัตโนมัติ ด้านล่าง dropdown:

```
┌─────────────────────────┐
│   แจ้งซ่อมเครื่องจักร   │
│      TY-001             │
│   จักรอุตสาหกรรม...    │
│   [  QR CODE IMAGE  ]   │
│  ผลิต / เย็บ F/C        │
│  🔲 สแกนเพื่อแจ้งซ่อม  │
└─────────────────────────┘
```

- **URL ที่ใช้ในการสแกน** จะแสดงด้านล่างกรอบ (readonly)

---

## ขั้นตอนที่ 6 — เพิ่มเข้าคิวพิมพ์

กดปุ่ม **"➕ เพิ่มในรายการพิมพ์"**

- ปุ่มจะเปลี่ยนเป็น **"✓ เพิ่มแล้ว!"** ชั่วคราว
- QR Code จะปรากฏในส่วน **"รายการ QR Code ที่จะพิมพ์"** ด้านล่าง
- ถ้าพยายามเพิ่มรายการซ้ำ ระบบจะแจ้งเตือน

### สร้าง QR หลายเครื่องพร้อมกัน

ทำซ้ำขั้นตอน 3–6 สำหรับเครื่องจักรเครื่องถัดไป ทุกรายการจะสะสมอยู่ในคิวพิมพ์

---

## ขั้นตอนที่ 7 — จัดการคิวพิมพ์

ในส่วน **"รายการ QR Code ที่จะพิมพ์"**:

| ปุ่ม | การทำงาน |
|------|-----------|
| **×** (มุมบนขวาของกรอบ) | ลบรายการนั้นออกจากคิว |
| **🗑 ล้างทั้งหมด** | ลบทุกรายการออกจากคิว |
| **🖨 พิมพ์ทั้งหมด** | พิมพ์ QR ทุกรายการในคิวพร้อมกัน |

---

## ขั้นตอนที่ 8 — พิมพ์ QR Code

1. กดปุ่ม **"🖨 พิมพ์ทั้งหมด"**
2. ระบบจะเตรียม QR Code ทุกรายการ (ขนาด 200×200) เป็นหน้าพิมพ์
3. กล่อง print dialog ของเบราว์เซอร์จะเปิดขึ้น
4. เลือก printer และกด **"พิมพ์"**

> **คำแนะนำการพิมพ์:**
> - ปิด "Headers and footers" เพื่อให้หน้าสะอาด
> - ตั้งขนาดกระดาษ A4, Landscape หรือ Portrait ตามจำนวน QR ที่สร้าง
> - ใช้กระดาษสติ๊กเกอร์เพื่อติดที่เครื่องจักรได้เลย

---

## ขั้นตอนที่ 9 — ติด QR Code ที่เครื่องจักร

ตัดและติด QR Code Label ที่ตัวเครื่องจักรในตำแหน่งที่มองเห็นได้ง่าย

---

## วิธีการใช้งาน QR Code (สำหรับผู้แจ้งซ่อม)

1. เปิดแอปกล้องหรือแอปสแกน QR บนโทรศัพท์
2. สแกน QR Code ที่ติดอยู่กับเครื่องจักร
3. เบราว์เซอร์จะเปิดฟอร์มแจ้งซ่อม โดยกรอกข้อมูลเหล่านี้ให้อัตโนมัติ:
   - ✅ รหัสเครื่องจักร
   - ✅ สาขา
   - ✅ ฝ่าย
   - ✅ หน่วยงาน
4. ผู้แจ้งกรอกเพิ่มเติมเฉพาะ **อาการเสีย** และ **ชื่อผู้แจ้ง**
5. กดส่ง

---

## ข้อมูลที่บรรจุใน QR Code

URL ตัวอย่าง:
```
https://example.com/pages/repair_form.php?machine=TY-001&branch=ASP&division=ผลิต&department=เย็บ F/C
```

| Parameter | ความหมาย |
|-----------|-----------|
| `machine` | รหัสเครื่องจักร |
| `branch` | สาขา |
| `division` | ฝ่าย |
| `department` | หน่วยงาน |

---

## คำถามที่พบบ่อย

**Q: เลือกฝ่าย/หน่วยงานแล้ว QR ไม่แสดง?**  
A: ต้องเลือกทั้งฝ่ายและหน่วยงานให้ครบทั้งคู่ ถึงจะแสดง preview

**Q: สร้าง QR เครื่องจักรหลายเครื่องในฝ่ายเดียวกันได้ไหม?**  
A: ได้ ระบบจำค่าฝ่าย/หน่วยงานล่าสุดไว้ให้อัตโนมัติ เพียงคลิกเครื่องถัดไปแล้วกด "เพิ่มในรายการพิมพ์" ได้เลย

**Q: พิมพ์แล้ว QR Code ไม่ชัด?**  
A: ตรวจสอบว่าปิดการ scale/shrink ใน print dialog แล้ว หรือลอง print เป็น PDF ก่อน

**Q: สแกน QR แล้วเข้าหน้าแจ้งซ่อมไม่ได้?**  
A: ตรวจสอบว่าโทรศัพท์มีการเชื่อมต่ออินเตอร์เน็ตและ URL ของระบบถูกต้อง
